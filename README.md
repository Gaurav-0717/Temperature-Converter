# Temperature Converter

This is a simple Temperature Converter web application built using **HTML, CSS, and JavaScript**.

## Features

* Convert temperature between **Celsius, Fahrenheit, and Kelvin**
* Input validation for numeric values
* Simple and user-friendly interface
* Convert and Clear buttons for easy operation

## How it Works

1. Enter the temperature value.
2. Select the unit you want to convert **from**.
3. Select the unit you want to convert **to**.
4. Click the **Convert** button to see the result.

## Technologies Used

* HTML
* CSS
* JavaScript
